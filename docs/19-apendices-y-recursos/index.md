# Módulo 19 — Apéndices y recursos

Material de consulta permanente: **la chuleta de comandos del curso, el mapa
hacia las certificaciones, glosario y bibliografía** para seguir aprendiendo.

## Contenido

### 19.1 — Chuleta de comandos del curso

- Referencia rápida de todos los comandos vistos, organizados por módulo y
  por tarea ("quiero buscar un archivo", "quiero ver qué ocupa el disco"...).
- Las 20 combinaciones de comandos más útiles del curso.

### 19.2 — Mapa de certificaciones

- **LPIC-1 / LPIC-2** (Linux Professional Institute): temario cruzado con
  los módulos de este curso.
- **RHCSA / RHCE** (Red Hat): qué practicar de más y entorno de examen.
- **CompTIA Linux+** y otras; cuál elegir según tu objetivo profesional.

### 19.3 — Laboratorios propuestos

- Laboratorio 1: servidor doméstico completo (NAS + servicios + backups).
- Laboratorio 2: pipeline DevOps de extremo a extremo.
- Laboratorio 3: endurecer y auditar un servidor expuesto.
- Ideas de proyecto personal para consolidar cada nivel.

### 19.4 — Glosario

- Términos del curso de la A a la Z, con referencias cruzadas al módulo
  donde se desarrollan.

### 19.5 — Bibliografía y recursos

- Libros de referencia: *The Linux Command Line*, *UNIX and Linux System
  Administration Handbook*, *How Linux Works*, *Systems Performance*.
- Documentación viva: Arch Wiki, man7.org, TLDP, documentación de cada
  distribución.
- Comunidades, listas de correo, podcasts y canales recomendados.
- Cómo seguir: especializaciones (seguridad, SRE, kernel, embebidos).

## Requisitos previos

Ninguno: es material de consulta para todo el curso.
