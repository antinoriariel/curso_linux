# Módulo 01 — Introducción al mundo Linux

Antes de tocar una sola tecla conviene entender **qué es Linux, de dónde
viene y por qué funciona como funciona**. Este módulo da el contexto
histórico y conceptual sobre el que se apoya todo el curso.

## Objetivos

- Entender qué es un sistema operativo, un kernel y una distribución.
- Conocer la historia de UNIX, GNU y Linux y la filosofía que los une.
- Saber elegir una distribución adecuada según el caso de uso.
- Comprender las licencias de software libre y de código abierto.

## Capítulos

### 1.1 — ¿Qué es un sistema operativo?

- Kernel, espacio de usuario y llamadas al sistema.
- Diferencia entre Linux (kernel) y GNU/Linux (sistema completo).
- Comparativa conceptual con Windows y macOS.

### 1.2 — Historia: de UNIX a Linux

- UNIX, BSD y la guerra de los UNIX comerciales.
- El proyecto GNU y Richard Stallman.
- Linus Torvalds y el nacimiento del kernel Linux (1991).
- POSIX y los estándares: qué significan en la práctica.

### 1.3 — La filosofía UNIX

- "Haz una cosa y hazla bien": programas pequeños y componibles.
- Todo es un archivo.
- Texto plano como interfaz universal.
- Cómo esta filosofía moldea las herramientas que usarás todo el curso.

### 1.4 — Licencias y software libre

- Las cuatro libertades del software libre.
- GPL, LGPL, MIT, BSD, Apache: diferencias prácticas.
- Software libre vs. código abierto vs. freeware.
- Modelos de negocio alrededor del software libre.

### 1.5 — Distribuciones: el mapa del ecosistema

- Qué compone una distribución: kernel, init, gestor de paquetes, escritorio.
- Las grandes familias: **Debian/Ubuntu**, **Red Hat/Fedora**,
  **Arch**, **SUSE**, **Gentoo**, **Alpine**.
- Modelos de publicación: versiones fijas (point release) vs. *rolling release*.
- Distros especializadas: servidores, seguridad (Kali), embebidos, contenedores.
- Criterios para elegir tu primera distribución.

### 1.6 — Dónde vive Linux hoy

- Servidores, la nube y los superordenadores.
- Android, sistemas embebidos e IoT.
- El escritorio Linux y el ascenso de WSL y SteamOS.
- La comunidad: wikis, foros, listas de correo y cómo pedir ayuda bien.

## Requisitos previos

Ninguno. Este módulo es el punto de partida del curso.
